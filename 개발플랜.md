## Phase 2 — 4대 서비스 구현 (2주)

### 현재 완성: 사주팔자 (월령선녀)
- 고도화 포인트: 결과 데이터를 `localStorage`에 "사용자 오행 프로필"로 저장
- 다른 서비스 진입 시 "사주에서 본 당신의 기질"을 참조하는 연결 멘트 추가

---

### 신규 구현 1: 타로카드 (루나)
**테마**: 무의식의 거울

**구현 흐름**:
1. 루나 캐릭터 인트로 애니메이션
2. 사용자가 고민 텍스트 입력
3. 카드 3장을 화면에서 직접 선택 (클릭/터치)
4. 카드 뒤집기 시 파티클 효과
5. **AI 이미지 생성 로딩** → 광고 노출 포인트
6. 3장 합성 "예언적 풍경화" 이미지 + Claude 해설 텍스트

**사주 연동 (일관성)**:
- localStorage에 사주 오행 프로필 있으면: "사주에서 화(火)의 기운이 강한 당신에게 이 카드는..."
- 이미지 생성 프롬프트에 오행 색상 반영

---

### 신규 구현 2: 손금 (카이로스)
**테마**: 인생 항로 지도

**구현 흐름**:
1. 카이로스 캐릭터 인트로
2. 카메라 접근 요청 (미지원 시 이미지 업로드 fallback)
3. 손 이미지 위에 캔버스 오버레이 — 금색 선 + 라벨 합성
4. "항로의 끝에 있는 보물을 확인하시겠습니까?" → **보상형 광고**
5. Claude API로 손금 해설 텍스트 생성

---

### 신규 구현 3: 이름풀이 (가온)
**테마**: 소리의 문장(Crest)

**구현 흐름**:
1. 가온 캐릭터 인트로 (붓글씨 애니메이션)
2. 이름 입력 → 획수/오행 분석 (한자 획수 DB or Claude 추출)
3. **AI 이미지 생성** → "당신의 가문 문장(Crest)" 생성 → 광고 노출
4. 결과 하단 "이 이름과 찰떡궁합인 행운의 성씨 찾기" 광고 연계

## Phase 3 — AI 이미지 생성 & 오행 일관성 (1주)

### 3-1. 이미지 생성 API 연동
추천: **fal.ai** (속도 빠름, 가격 합리적) 또는 **OpenAI DALL-E 3**

`api/image.js` Vercel 서버리스 함수:
```js
// POST /api/image
// body: { prompt, style }
// → { imageUrl }
```

### 3-2. 오행별 이미지 프롬프트 설계
> **기획 핵심 1번 구현**: 사주의 지배 오행 → 모든 서비스 이미지에 통일된 색감 적용

| 오행 | 주조색 | 이미지 키워드 |
|---|---|---|
| 🔥 화(火) | 붉은색, 주황, 금 | 태양, 봉황, 용암, 황혼 |
| 🌳 목(木) | 초록, 청록 | 울창한 숲, 용, 봄비, 신목 |
| 💧 수(水) | 파랑, 검청 | 달, 심해, 안개, 파도 |
| ⚙️ 금(金) | 흰색, 은빛 | 별, 검, 서리, 설원 |
| 🌍 토(土) | 황금, 갈색 | 대지, 산, 고성, 곡식 |

사용자 오행 프로필이 없으면 중성 프롬프트 사용.

### 3-3. 광고 연동 타이밍
```
AI 이미지 생성 요청 → [광고 5초 노출] → 이미지 완료 표시
```
"AI가 이미지를 생성 중입니다 (약 5초)" 문구로 자연스럽게 처리.

## Phase 4 — 광고 & 공유 카드 기능 (1주)

### 4-1. Google AdSense 연동
- **메인 화면 하단**: 띠 배너 (320×50)
- **분석 로딩 화면**: 인터스티셜 광고 (전면) — 체류 시간 가장 긴 구간
- **결과 화면 하단**: 문맥 맞춤형 광고 (결과 직후 배치)
- **손금 최종 결론**: 보상형 동영상 광고 (시청 후 잠금 해제)

```jsx
// components/ads/InterstitialAd.jsx
// components/ads/BannerAd.jsx
// components/ads/RewardAd.jsx
```

### 4-2. SNS 공유 카드 생성
> **기획 핵심 2번 구현**: 인스타그램 스토리 최적화 공유 유도

- Canvas API로 결과 이미지 카드 생성 (1:1 비율, 1080×1080px)
- 구성: AI 생성 이미지 + 캐릭터 로고 + 결과 한 줄 요약
- 버튼:
  - **"내 수호령 다운로드"** (사주/타로 결과)
  - **"내 인생 지도 저장"** (손금 결과)
  - **"내 문장(Crest) 저장"** (이름풀이 결과)

```jsx
// components/share/ShareCard.jsx
// utils/shareCardGenerator.js  ← html2canvas 또는 Canvas API
```
